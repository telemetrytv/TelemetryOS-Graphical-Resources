Print Assets

High-resolution 300 DPI files for professional printing

Print Quality Guidelines

  • Resolution: All print assets are 300 DPI for sharp reproduction
  • Color Profile: Use CMYK color values for professional printing (see brand guidelines)
  • File Format: PNG for universal compatibility, WebP for digital previews
  • Sizing: Use largest available size and scale down as needed
  • Color Selection: Amber for branded materials, Black for professional/monochrome, White for dark backgrounds
  • Aspect Ratio: Logos are 1:1 (square), Wordmarks are 4:1 (horizontal)

CMYK Color Values for Print

  • Amber (Primary): C:0% M:28% Y:79% K:3% — Use for branded color printing
  • Black: C:0% M:0% Y:0% K:100% — Use for professional/monochrome contexts
  • White: C:0% M:0% Y:0% K:0% — For dark backgrounds or spot color printing
