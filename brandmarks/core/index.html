Core Brand Marks

Source of truth • All platform assets derive from these master files

Usage Guidelines

  • Logomarks: Square format (1:1 ratio) for app icons, avatars, and square placements
  • Wordmarks: Horizontal format (4:1 ratio) for headers, banners, and wide spaces
  • SVG First: Use SVG when possible for scalability and smallest file size (97%+ browser support)
  • WebP + PNG: Use WebP with PNG fallback for optimal performance in raster images
  • Colors: Amber (#F8B334) on light backgrounds, White on dark backgrounds, Black for print/professional
